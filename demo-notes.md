# Demo Notes

## What The Demo Should Show

The demo should show a ChatGPT shared link being converted into a structured Markdown handoff file that is ready to paste into an AI coding tool.

## Step-by-Step Demo Flow

1. Open the local app.
2. Paste one valid ChatGPT shared link.
3. Click `Generate Context Markdown`.
4. Show the loading state.
5. Scroll the Markdown preview.
6. Copy the Markdown.
7. Download `context-handoff.md`.
8. Paste a second link and generate one combined handoff.

## Edge Cases Worth Showing

- Empty textarea.
- Invalid non-ChatGPT link.
- Multiple links where one fails and one succeeds.
- Missing `OPENAI_API_KEY`.

## Screenshot Or Video Ideas

- Before state with shared links pasted.
- After state with generated Markdown preview.
- Downloaded `context-handoff.md` opened in an editor.

## Short Demo Script

- "Stop losing context when you move from ChatGPT to Codex."
- "I built a tool that turns ChatGPT planning chats into coding briefs."
- "One shared link in, one clean Markdown handoff out."
